Background animation demo - анимированный фон на форме с AlphaControls

acBGAnimationDemoExe - пример создания анимированного фона.

MatrixStyle - пример с использованием скина Terminal4bit и The Matrix Screen Saver за авторством Andy Kern (http://cc.embarcadero.com/item/15103)

Известные проблемы:
 - для скинов Afterburner и NeonNight анимация не работает
 - долгая загрузка из-за очень больших текстур в TsAlphaImageList 
 - в момент когда происходит обновление формы иногда можно нажать на то что лежит под ней
 - форма развернутая на весь экран при слишком большой скорости анимации может создавать нагрузку на процессор до 50%
